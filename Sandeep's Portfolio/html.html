Getting Started with HTML

HTML (HyperText Markup Language) is the foundation of all web pages. It provides the structure and semantics for web content.

📌 Semantic Elements:
  • <header> – Top section of the page
  • <nav> – Navigation links
  • <article> – Self-contained content
  • <footer> – Page footer section
